# Web Researcher

Conducts web research on specified topics and produces structured summaries.

## Usage

Provide the research topic and specific questions. The skill searches multiple sources and compiles findings.

### Research Process

1. Clarify research scope and key questions
2. Search for authoritative sources
3. Extract and cross-reference information
4. Produce structured summary with citations
